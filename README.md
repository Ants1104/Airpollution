## 目的
減緩工業及畜牧農業所造成的空污問題

## 動機
由於近年來在新聞上看到許多養殖畜農牧業所造成的排泄物及工廠排廢棄物，
其實是PM2.5人為造成的主因，又因民間政府在檢驗這一塊缺乏立即性。
因此本團隊為了解決此問題，本團隊構思了一套民眾參與式的舉報系統，
想藉由民眾在第一時間所觀察到的氣味去建立時間點、地點進而分析污染的源頭。

## 資料集
空氣盒子
行政院農委會畜牧登記管理系統
經濟部統計處工廠名錄
內政部地籍圖資
網路便民服務系統
